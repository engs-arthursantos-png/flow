Atividades do Desenvolvedor Adrian

Cadastro de usuários:
um sistema para usuarios se cadastrarem;

Definição da funcionalidade de registro de novos usuários:
um sistema para salvar as informaoes de cadstro;

Login de usuários:
um sistema para ususarios entrarem na sua conta que foi registrada;

Definição da autenticação de acesso:
um sistema que verifique se as informaçoes de cadastro estao corretas;

Cadastro de músicas:
um sistema para adicionar as  usicas no aplicativo;

Organização das músicas cadastradas:
um sistema que organize as musicas atraves de filtros;

Cadastro de artistas:
um sistema onde os artistas e criadores das musicas possam se cadastrar;

Organização dos artistas cadastrados:
um sistema que organize os artistas por filtros como o genero da musica de certo artista;

Sistema de pesquisa de músicas:
um sistema para que o  ouvinte pesquise a musica e os filtros automaticamente encontre musicas relacionadas a pesquisa;

Busca de músicas pelo nome: 
um sistema que encontre automaticamente a musica pesquisada que estiver registrada;

Reprodução de músicas:
um sistema para reproduzir a musica em qualquer dispositivo;

Execução das músicas selecionadas:
um sistema que execute a musica que o ouvinte selecionou;

Pausar reprodução:
um botao que quando for apertado, pare a musica na parte que ela estava sendo reproduzida;

Interrupção temporária da música:
um sistema que interrompa temporariamente a musica e faça ela voltar do momento que foi interrompida, como por exemplo os anuncios do spotfy;

Próxima música:
um botao que quando for apertado pule para o começo da proxima musica;

Avanço para próxima faixa:
um sistema que avançe a faixa quando selecionado;

Música anterior:
um botao que qunado for acionado volte para o começo da ultima musica reproduzida;

Retorno à faixa anterior:
um botao que quado for acionado volte para faixa mais recente que o ouvinte estava;

Controle de volume:
um sistema que tenha numeros de 0 a 100%, o quanto mais proximo de 0 mais baixo sera o som, o quanto mais proximo de 100 mais alto sera o som;

Ajuste do volume do player:
um sistema que ajuste o voliume do player entre 0 a 100;

Criar playlists:
uma aba no aplicativo onde o ouvinte possa selecioar varias musicas desejadas e move-las para uma pasta separada, onde so serao reproduzidas as musicas desta pasta;

Criação de listas personalizadas:
um sistema que o ouvinte possa criar uma lista personalizada da forma que ele desejar

Editar playlists:
um sistema em que o ouvinte possa editar a sua playlist antes de criar;

Alteração de playlists existentes:
um sistema em que o ouvinte possa editar a playlist que ja foi criada;

Favoritar músicas:
um sistema em que o ouvinte possa favoritar uma musica e essa musica fique em uma pasta separada apenas para musicas favoritas;

Marcação de músicas favoritas:
um sistema em que as musicas que forem favoritadas fiquem com uma marcaçao indicando que foi favoritada;
